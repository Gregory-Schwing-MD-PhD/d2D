# &delta;2D
Deriving Secondary Structure Populations from NMR chemical shifts

## Install
- clone the repository
- in `src` run `make` and then
- export CAMDBV3=`the d2D base folder`

## Run it
./src/d2D+.x -file test/test.txt -shifty
